GAME SUMMARY

Hopebringers is a tactics game with a medieval fantasy setting. The main idea of the game was to have a tactics game with a tabletop game feeling and perfect information (as an experiment we made all the skills and enemies behave in a deterministic way without randomness). The player has 8 characters with different skills and upgrades. Each character has a unique skill set and utilities. For example, the knight uses a shield both to attack and defend himself so he doesn't receive damage from the front and all of its upgrades revolve around these concepts.

The game consists of 25 levels and 3 extra levels that can be replayed and are randomly generated. Between each level, the characters will have conversations advancing the plot forward and developing their relationships.
