GAME SUMMARY

Icarus is an adventure puzzle game, and it was my biggest project at the moment and the first time working with a team. The story is set on a spaceship traveling from Earth to Mars. The main character will need to stop the ship's AI which is trying to take control of the crew.

Mechanically the game is played by talking with NPCs, solving puzzles, and finding, combining, and using items.

Here is the Itchio page where you can download the game for free. Unfortunately, the game is only available in Spanish.
